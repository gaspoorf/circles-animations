# Circles Scroll-Driven Animation

This half-circle interaction explores movement, rhythm, and spatial navigation.
A scroll-driven interaction built around half-circle shapes, exploring motion, rhythm, and spatial navigation through scrolling.

## Installation

Install dependencies:

```bash
npm i
```

## Development

Start the development server:

```bash
npm run dev
```

## Tech Stack

- Vanilla JavaScript
- GSAP
- Vite
